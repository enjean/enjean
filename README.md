## 🛠 Featured Work

### 🏀 [March Madness Bayesian Optimizer](https://github.com/enjean/bracket)
**Stack: Kotlin 2.3.0, Bayesian Statistics, Beam Search**

A decision-support tool to help me fill out March Madness brackets using a variety of algorithms. 
* Implemented a recursive tree-based optimization engine using a **Beam Search** pruning strategy that solves the $2^{63}$ combinatorial explosion of tournament brackets.
* Features a decoupled **Strategy Pattern** for interchangeable win-probability and scoring models.

### 🌲 [Cabin Tracker](https://github.com/enjean/cabin-log) (Work in Progress)
**Stack: Kotlin, Spring Boot, PostgreSQL, GCP**
A personal project for tracking statistics about the number of visits per person to a cabin (or other second home). This serves as my playground to use the Spring Boot patterns I have learned in my professional life on a project of personal interest.
